シーケンス図（処理フロー）
```plantuml
@startuml
title テンプレ
'==ライフライン=='
actor User <<Actor>> #ff8b8bff
participant 役割名_分子名 as temp #8ea2fcff
participant UserInterface as UI #8ea2fcff

box "Backend" #E8F5E9
  participant 認証 as Auth
  participant App
end box

box "DataBase" #E8F5E9
  participant DB
end box


'==END ライフライン=='

== メモ ==
group グルーピング [コメントの位置設定Temp]
  note left of UI #aqua
  "note left of UI #aqua"
  end note

  rnote right of UI: "rnote[四角] right of UI:"

  hnote over UI: "hnote[六角] over UI:"
end

note over UI: "/"で横ならべ
  / note over Auth: " / note"

note over UI, Auth #FFAAAA: 画面の\n UIと認証の上.

note over UI, DB
改行"\n"いらない
認証とUIの上に表示
end note

== メッセージ ==

' シーケンス図確定用（表示されてもOK）
User -> User : init

User -[#57db45]-> UI : ログイン操作
UI -[#db4242ff]> Auth : 認証リクエスト
activate UI

Auth -> DB : ユーザー取得
activate DB
DB --> Auth : ユーザー情報(リプライメッセージ)
destroy DB
Auth --> UI : 認証結果

deactivate UI

alt 認証成功
  UI -> App : 業務処理要求
  App -> DB : データ取得
  DB --> App : データ
  App --> UI : 処理結果
else 認証失敗
  UI --> User : エラー表示
end

UI --> User : 画面表示
@enduml

```

図③：構造・責務整理図（クラス図）

```plantuml
@startuml

'==全体デザイン=='
skinparam backgroundColor #FAFAFA
skinparam shadowing false
skinparam defaultFontName Meiryo
skinparam defaultFontSize 12

skinparam roundCorner 12 '角の柔らかさ'
skinparam dpi 200 '解像度'
skinparam padding 5 '図形内のレイアウト'
skinparam margin 10 '図形外のレイアウト'

'横に並べる'
left to right direction


package "Frontend" {
  class WebUI <<Boundary>>
}

package "Backend" {
  class AuthService <<Control>>
  class ApplicationService <<Control>>
}

package "Database" {
  class UserTable <<Entity>>
}

WebUI --> AuthService
WebUI --> ApplicationService
AuthService --> UserTable
ApplicationService --> UserTable
@enduml
```
